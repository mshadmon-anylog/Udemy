What is TypeScript: 
    * A JavaScript (JS) superset -- it takes JS and adds ontop of it 
    * It cannot be executed like JS 
    * Programming Langugage + Compiler 
    * Extra error checking instead of runtime errors

Why Use TypeScript: 
    * Extra error checking instead of runtime errors
    * validate code
